JavaScriptの基本構文と特徴

はじめに：JavaScriptとは？

JavaScriptは、ウェブページに動的な機能や対話性を追加するための代表的なプログラミング言語です。

もともとはブラウザ上で動作するクライアントサイドの言語として開発されましたが、現在ではNode.jsによってサーバーサイドの開発にも使用されるなど、用途の幅が大きく広がっています。

HTMLがページの構造を定義し、CSSがその見た目を整えるのに対し、JavaScriptはユーザーとのインタラクションやデータの処理、非同期通信などを担う、まさに“動き”の部分を支える技術です。

基本的な構文

JavaScriptを使いこなすためには、まずはその基本構文を理解することが重要です。ここでは、変数宣言・データ型・制御構文・関数という、基礎中の基礎を丁寧に見ていきます。

1. 変数宣言

JavaScriptでは、letやconstを使って変数を宣言します。varという旧来の方法もありますが、現在ではあまり使われません。

  • let: 値の再代入が可能な変数。
  • const: 一度代入した値を変更できない定数。
// let の例
let message = "こんにちは！";
message = "さようなら！"; // 再代入可能

// const の例
const PI = 3.14;
// PI = 3.14159; // これはエラーになります

2. データ型

JavaScriptには、以下のような主なデータ型があります。それぞれの型はtypeof演算子で確認することができます。

  • Number: 数値（整数や浮動小数点）
  • String: 文字列
  • Boolean: 真偽値（true / false）
  • Object: 複数の値をまとめて扱える構造
const age = 30;
const name = "田中";
const isMember = false;
const user = { name: "田中", age: 30 };

3. 制御構文（if文とforループ）

条件に応じて処理を分岐させたり、繰り返し処理を行うには、if文やfor文を使います。 

// 条件分岐
const score = 90;
if (score >= 80) {
  console.log("合格！");
} else {
  console.log("不合格");
}

// ループ処理
for (let i = 0; i < 3; i++) {
  console.log(i + 1 + "回目のループ");
}

関数：処理のまとまり

関数とは、何度も使いたい処理を1つにまとめて再利用可能にしたものです。コードの可読性や保守性を高めるのに役立ちます。 

// 通常の関数定義
function greet(name) {
  return name + "さん、こんにちは！";
}

const greeting = greet("佐藤");
console.log(greeting);

アロー関数（Arrow Function）を使うと、より短く記述できます：

// アロー関数
const add = (a, b) => a + b;
const multiply = (a, b) => a * b;

console.log(add(2, 3));      // 5
console.log(multiply(4, 5)); // 20

WebSocket：リアルタイム通信を可能にする技術

WebSocketは、サーバーとクライアントの間で双方向のリアルタイム通信を実現するプロトコルです。HTTPとは異なり、接続を一度確立するとサーバーからのプッシュ通知も可能になり、チャットアプリやゲーム、株価更新などに最適です。

基本的な使い方（クライアント側）

// WebSocketサーバーへの接続
const socket = new WebSocket("wss://example.com/socket");

// 接続が開いたとき
socket.addEventListener("open", () => {
  console.log("接続されました");
  socket.send("Hello Server!");
});

// メッセージを受信したとき
socket.addEventListener("message", (event) => {
  console.log("サーバーからのメッセージ:", event.data);
});

// 接続が閉じられたとき
socket.addEventListener("close", () => {
  console.log("接続終了");
});

WebSocketは非同期処理が基本なので、イベントリスナーによって接続・受信・切断などを監視します。リアルタイム性が求められる現代のアプリ開発において非常に重要な技術です。

まとめ

本稿では、JavaScriptの基本構文である変数宣言、データ型、制御構文、関数について体系的に解説するとともに、リアルタイム双方向通信を可能にするWebSocketの技術的概要も紹介しました。JavaScriptはクライアントサイドだけでなくサーバーサイドでも広く活用されており、その柔軟性と表現力の高さから、モダンなウェブアプリケーション開発において不可欠な言語となっています。基礎概念を確実に理解した上で、非同期処理やイベント駆動型プログラミングなどの高度な技術へと応用していくことが、効率的かつ堅牢なシステム構築の鍵となります。
